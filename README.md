# 🌐 Odin Landing Page

A responsive landing page built from scratch as part of [The Odin Project – Foundations](https://www.theodinproject.com/lessons/foundations-landing-page) curriculum.

---

## 📋 About the Project

This project was built by following a provided design mockup — two reference images showing the full layout and typography/color details. The goal was to recreate the design using only HTML and CSS, without using any frameworks or libraries.

Key constraints and goals:

- Match the overall visual structure of the provided design
- Get elements positioned correctly relative to one another (pixel-perfect was **not** required)
- Substitute custom content, colors, or images if desired

---

## 🛠️ Built With

- **HTML5** — semantic markup
- **CSS3** — flexbox layout, custom properties, responsive styling

---

## 🚀 Getting Started

No build tools required. Just clone and open.

```bash
git clone https://github.com/your-username/odin-landing-page.git
cd odin-landing-page
open index.html
```

Or simply open `index.html` in your browser.

## 🎯 What I Practiced

- Building a multi-section page layout from a visual spec
- Using **Flexbox** for navigation, hero, cards, and footer sections
- Applying font choices and color palettes from a design reference
- Searching for CSS properties I hadn't used before (e.g. `border-radius`, `box-shadow`)
- Writing clean, structured HTML with appropriate semantic tags
